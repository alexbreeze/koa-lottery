======

获取实时彩票结果的开源API

## 功能列表
+ 重庆时时彩
+ 其他彩票接口

## Installation

```sh
$ npm install lottery
```

## Use with koa

```js
var wechat = require('lottery');

```
备注：token在微信平台的开发者中心申请
